ABOUT

    The website is designed for the use of elementary students, which allows users to have fun while learning, without giving them any difficulty in regards to use and navigation.
   This project focuses on science- trivias, fun facts, and basic concepts about animals. The portable encyclopedia is not only limited to giving information; we also have a quiz for every topic to make sure that our users gained at least a little bit of knowledge from our project.
   The proponents are expected to develop this website using basic coding elements and tags of HTML and CSS. the website will observe a minimalist yet interactive interface in order to make it look clean. The proponents are expected to finish this project within 12 weeks.
